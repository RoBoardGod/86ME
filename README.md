86ME (86Duino Motion Editor) Mk-II v2.1.0
---------
The 86ME Mk-II allows that you edit servo action frames to be used by the Servo86 library, and can automatically generate 86Duino sketches to control motion of servos

#### INSTALLATION ####

1. System reqirement: Windows XP/Vista/7/8/8.1/10 with .NET framework 4.5.
2. Download [86ME](https://github.com/Sayter99/86ME/releases/download/86ME/86ME_v2.0.zip).
3. Unzip 86ME and execute 86ME.exe to start.
